# Genetic Variant Classification with XGBoost

Bu proje, genetik varyant verileri üzerinde patojenik ve benign varyantları sınıflandırmak amacıyla hazırlanmış bir makine öğrenmesi çalışmasıdır.

## Proje Amacı

Bu çalışmada veri ön işleme, özellik mühendisliği, sınıf dengesizliği yönetimi, model eğitimi ve model değerlendirme adımları uygulanmıştır.

## Kullanılan Teknolojiler

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
- SHAP

## Uygulanan Adımlar

- Veri temizleme
- Eksik değer analizi
- Kategorik değişken dönüşümü
- Train-test split
- XGBoost ile sınıflandırma modeli geliştirme
- GridSearchCV ile hiperparametre optimizasyonu
- Threshold analizi
- Confusion matrix ve sınıflandırma metrikleri
- SHAP ile model yorumlanabilirliği

## Model Değerlendirme

Model performansı accuracy, precision, recall, F1-score, ROC-AUC, PR-AUC ve balanced accuracy metrikleriyle değerlendirilmiştir.
